# Projeto Next.js com CI/CD (GitHub Actions + Vercel)

Este repositório contém uma aplicação **Next.js** configurada com uma pipeline completa de **CI/CD**, incluindo:

- ✔ Validação de código com **ESLint**
- ✔ Execução de testes automatizados
- ✔ Build automático da aplicação
- ✔ Deploy automático para a **Vercel** a cada push na branch `main`

---

##  Tecnologias Utilizadas

- **Next.js**
- **React**
- **Node.js**
- **GitHub Actions**
- **Vercel**

---

##  CI – Integração Contínua

A pipeline de CI executa automaticamente a cada **push** ou **pull request** na branch `main`.

Etapas:

- Instalação das dependências (`npm ci`)
- Verificação de lint (`npm run lint`)
- Execução de testes (`npm run test`)
- Build da aplicação (`npm run build`)



## ▶ Como rodar o projeto localmente

npm install
npm run dev

Acesse:

http://localhost:3000
